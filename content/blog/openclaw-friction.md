The first promise of software agents is relief. The second, more difficult promise, is judgment.

When an interface claims autonomy but asks the user to supervise every fragile step, the result is not collaboration. It is a new kind of clerical labor wearing a futuristic label.

Friction is rarely just a bug. It is a design argument made visible: a trace of where the system still needs a human to carry its uncertainty.

## Notes

- Autonomy should reduce cognitive switching.
- Approval flows need clear stakes.
- A good agent explains uncertainty without making the user manage its anxiety.
