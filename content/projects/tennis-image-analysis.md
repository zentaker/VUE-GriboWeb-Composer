The Tennis Image Analysis Pipeline begins with a simple question: what remains in an image after the point has already moved on?

The project treats footage as a technical trace. Court lines, body position, timing, and tactical pressure become signals that can be extracted, reviewed, and eventually explained.

## Current shape

The project is represented here as a repository record and technical docs scaffold. Real processing services are intentionally out of scope for this stage, but the dossier already records the shape of the future system.

## Future documentation

- Data preparation and frame sampling.

- Annotation workflow and interface review.
