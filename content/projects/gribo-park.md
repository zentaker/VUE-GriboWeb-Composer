Gribo Park is a sketch for browsing memory as a place.

It is not a product implementation yet. It is a vocabulary for shelves, paths, fragments, rooms, and the feeling of moving through an archive that is still alive.

The project asks what a repository becomes when it stops behaving like a list and starts behaving like a small cultural landscape.
